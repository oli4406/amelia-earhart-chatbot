# Amelia Earhart Chatbot

This is a Chatbot with the persona of Amelia Earhart, with the primary function of giving requested flight information to the user.

>[NOTE]
>For the login system to function, MongoDB will need to be installed on your device.
>Once installed run 'npm start' or 'npm run dev' to simultaneously start the frontend and backend.

## Defaults:
Frontend    -   http://localhost:5173/
Backend -   http://localhost:3000/

>[IMPORTANT]
>Remember to add SERPAPI_API_KEY and GEMINI_API_KEY to .env
